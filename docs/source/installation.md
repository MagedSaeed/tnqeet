Installation
============

**tnqeet** supports Python >= 3.8.

## Installing with `pip`

**tnqeet** is available [on PyPI](https://pypi.org/project/tnqeet/). Just run

```bash
pip install tnqeet
```

## Installing from source

To install **tnqeet** from source, first clone [the repository](https://github.com/MagedSaeed/tnqeet):

```bash
git clone https://github.com/MagedSaeed/tnqeet.git
cd tnqeet
```

Then run

```bash
pip install -e .
```
